例题1 在z轴上求两点A(3，1，-4)和B(5，3，2)等距离的点。

解：设z轴上点为M(0, 0, z)，由题意|MA| = |MB|，即

​ $$\sqrt{(3-0)^2 + (1-0)^2 + (-4 - z)^2} = \sqrt{(5-0)^2 + (3-0)^2 + (2-z)^2}$$

解得，z = 1，所求点位为M(0,0,1)。
